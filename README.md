# nhis-macro-core
건보 청구용 매크로 공용 르포

## Project Overview 
`nhis-macro-core`는 **보험 청구 업무를 보조하기 위한 반자동(semi-automated) 워크플로우 프레임워크**입니다.  
이 저장소는 전체 처리 흐름을 제어하는 **워크플로우 구조, 단계 인터페이스, 실행 경계**만을 정의하며,  
실제 시스템 조작이나 자동 입력 로직은 포함하지 않습니다.

본 프로젝트는 특정 기관·시스템에 종속되지 않는 **아키텍처 참고용 코어 레이어**를 제공하는 것을 목표로 합니다.

## Design Philosophy
본 프로젝트는 다음 원칙을 전제로 설계되었습니다.

- **반자동(Human-in-the-loop) 구조**
  - 모든 핵심 처리 흐름에는 사용자의 명시적 확인 단계가 포함됩니다.
  - 최종 전송 또는 확정 행위는 반드시 사용자의 직접 동작으로 이루어집니다.

- **책임 분리**
  - 본 코어 프레임워크는 판단·결정·전송의 책임을 대신하지 않습니다.
  - 실제 업무 결과에 대한 책임은 전적으로 사용자에게 있습니다.

- **경계 기반 설계**
  - 워크플로우 제어와 실제 자동화 구현을 명확히 분리합니다.
  - 시스템별 구현, 규칙, 노하우는 코어 레이어에 포함되지 않습니다.

---

## What This Repository Contains
- 워크플로우 오케스트레이션 구조 (상태 흐름, 단계 전이)
- Step, Validator, Automation 등에 대한 **추상 인터페이스**
- 실행 흐름 제어용 러너(Runner) 구조
- 설계 및 아키텍처 관련 문서

---

## What This Repository Does NOT Contain
- 실제 자동 입력(매크로) 스크립트
- 특정 보험 시스템 또는 외부 시스템에 대한 조작 로직
- 실무용 청구 규칙, 검증 규칙, 필드 매핑 정보
- 계정 정보, 인증 정보, 개인정보 처리 로직
- 즉시 사용 가능한 배포용 애플리케이션

이 저장소의 코드만으로는 **실제 보험 청구 업무를 수행할 수 없습니다.**

---
